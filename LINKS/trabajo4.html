    Tecnológico de Estudios Superiores de Chalco

    Desarrollo de un sistema omnicanal para la administración de campañas telefónicas en la empresa Bonum S.A.PI. de C.V.

    Autor: Ojeda Amaro, Ismael

    Fecha de publicación:2022-04-01

    Editorial: Tecnológico Nacional de México

    Institución: Tecnológico de Estudios Superiores de Chalco

    Descripción

    Las empresas que se enfocan en brindar servicios de telemarketing buscan procesos de creación de campañas cortos e intuitivos los cuales permitan un mayor alcance de clientes, dejando atrás sistemas carentes de experiencia de usuario favorable. El objetivo de este estudio es crear un sistema omnicanal para la generación de campañas telefónicas enfocado en administrar las campañas de telemarketing y sus componentes desde un solo sistema web a partir de la descripción y estudio de procesos ya existentes en el sistema Vicidial. El uso de un sistema omnicanal que permita la unión de MOR16 y Vicidial, reducirá el número de tareas realizadas por el usuario cliente o el usuario administrador. El tiempo en la creación de campañas se reducirá a treinta minutos por campaña, gracias al uso del sistema omnicanal dedicada a la gestión de campañas telefónicas Como resultado se generó un sistema omnicanal implementando tecnologías como Laravel, el nuevo sistema web permite administrar los componentes requeridos por una campaña de telemarketing capaz de agilizar el proceso de la creación de cam-pañas telefónicas, el cual fue comprobado cuando personal capacitado en la creación de campañas probó el nuevo sistema omnicanal....
